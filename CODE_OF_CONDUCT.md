# Code of Conduct

## Our Standard

Be respectful, direct, and technically honest.

Examples of positive behavior:

- assuming good intent
- giving actionable technical feedback
- disagreeing without being dismissive
- focusing on the problem instead of the person

Examples of unacceptable behavior:

- harassment or personal attacks
- discriminatory language
- intentionally disruptive behavior
- publishing private information without permission

## Enforcement

Project maintainers may remove comments, issues, pull requests, or contributions that violate this code of conduct.

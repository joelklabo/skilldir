skilldir

One ordinary skills folder. Ordered sources. First source wins.

skilldir scans multiple skill trees and materializes the resolved set as normal symlinks. Existing agent harnesses can keep reading a plain directory without any plugin, wrapper, or custom protocol.

Config

{
  "sources": [
    "/home/honk/code/project/.agents/skills",
    "/home/honk/.codex/skills",
    "/home/honk/.claude/skills"
  ],
  "output": "/home/honk/.agents/skills"
}

Status Output

playwright -> /home/honk/.codex/skills/playwright
  shadowed:
  - /home/honk/.claude/skills/playwright

docs -> /home/honk/code/project/.agents/skills/docs

Compatibility

The generated output can target ~/.agents/skills, ~/.claude/skills, or any other ordinary directory that an agent harness already reads.
